1//! # fugue-causal: Bayesian Causal Inference via Generalized Bayes
2//!
3//! Extends the fugue probabilistic programming library with loss-based causal inference
4//! using generalized (Gibbs) posteriors.
5//!
6//! **Source Paper:** Javurek, E., et al. (2026). "Generalized Bayes for Causal Inference."
7//! ArXiv:2603.03035v1. <https://arxiv.org/abs/2603.03035>
8//!
9//! ## Core Idea
10//!
11//! Instead of specifying full data-generating models P(X, A, Y | ξ) and placing priors on
12//! nuisance components, we:
13//!
14//! 1. Place priors directly on causal estimands θ (e.g., ATE, CATE)
15//! 2. Update via identification-driven loss functions (not likelihoods)
16//! 3. Use Gibbs posteriors: q(θ | D) ∝ exp{−ωn·L_n(θ)} · π(θ)
17//!
18//! **Key advantage:** Formal robustness. Neyman-orthogonal losses (like DR/AIPW) give
19//! second-order nuisance robustness: TV divergence = O_P(√n · r_n²), not O_P(√n · r_n).
20//!
21//! ## Example
22//!
23//! ```ignore
24//! use fugue_causal::*;
25//!
26//! let problem = CausalProblem {
27//!     estimand_prior: prior_ate(),
28//!     identifier: DoublyRobust,
29//!     nuisance_estimator: Box::new(CausalForest::new()),
30//!     folds: 5,
31//! };
32//!
33//! let posterior = infer_causal(problem, observational_data)?;
34//! ```

51/// A causal inference problem specification.
52pub struct CausalProblem<T: CausalIdentifier> {
53    /// Prior distribution over the causal estimand: log π(θ)
54    pub estimand_prior: Box<dyn Fn(f64) -> f64>,
55    /// Strategy for identifying causal effects from observational data
56    pub identifier: T,
57    /// Estimator for nuisance components (propensity, outcomes, etc.)
58    pub nuisance_estimator: Box<dyn NuisanceEstimator>,
59    /// Number of folds for cross-fitting
60    pub folds: usize,
61}
